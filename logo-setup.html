La Empresa Logo Setup

To complete the setup of your website with the La Empresa brand, you need to save the logo images to the correct location.

Light Background Version

Right-click on this image and select "Save Image As..."

Save as: public/assets/images/logo-light-bg.png

Dark Background Version

Right-click on this image and select "Save Image As..."

Save as: public/assets/images/logo-dark-bg.png

Step-by-Step Instructions

  1. Right-click on each logo image above and select "Save Image As..."
  2. Navigate to your project's public/assets/images directory
  3. Save the light background version as logo-light-bg.png
  4. Save the dark background version as logo-dark-bg.png
  5. Refresh your website to see the logos in place

Note: The website is already configured to use these logos in both dark and light themes.
